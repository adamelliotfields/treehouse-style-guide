## Installation

```bash
git clone --single-branch -b dev https://github.com/adamelliotfields/treehouse-style-guide.git
```

## Development

```bash
# Install dependencies.
npm install

# Run the live reload server.
npm start

# Build assets without running the server.
npm run build
```
